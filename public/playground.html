LITENAI PLAYGROUND


Available on demand. Please contact us to try our early access program or for any information.

LitenAI offers different types of interfaces to chat with data to observe, reason and visualize. Liten copilot orchestrates the actions using a super agent based on customer prompt with an appropriate workflow of data and AI agents.

  • LitenAI standalone chat acts as a smart data copilot. Users can observe data by asking query questions. They can also reason it out using natural languages.
  • LitenAI can be installed in a jupyter notebook using python pip. Liten provides a single API call to do all your reasoning. You can try it out in Liten provided sample python notebook.
  • LitenAI is available as a chat bot for slack. You can add @Liten as a bot in your slack, and chat to it to get answers for your data.
  • LitenAI is working on a splunk application.
